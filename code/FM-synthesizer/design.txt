
FM. BISON synthesizer design.

Features for version 0.1:

[X] - 1 hardcoded tone (patch).
[X] - Polyphony of 8; no voice stealing.
[X] - 1 global ADSR which modulates amplitude.
[X] - 1 simple filter for low-pass cutoff and resonance.
[X] - Controls for: filter, master drive, modulation, oscillator type (FIX!) & ADSR.

Features for 0.2:

[X] - More elaborate filter (MOOG) plus wet<->dry mix.
- Reverse modulation ratios.

Features for 0.3:
- Voice stealing.
- First (hardcoded) patch, emulating a well known instrument (TOTO brass!).
